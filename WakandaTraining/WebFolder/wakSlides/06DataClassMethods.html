Dataclass Method

  • Executes when called
  • Can accept parameters and return a value
  • Always executed server-side
  • Can be called from the browser if allowed

  • Three levels of class methods
    • Class
      Collection
      Entity

Dataclass Method Levels

  • Class level method
    • Becomes a property of the dataclass
      Called using a dataclass reference
      The keyword this refers to the dataclass inside the method
  • Collection level method
    • Becomes a property of entity collections from the dataclass
      Called using an entity collection
      The keyword this refers to entity collection inside the method
  • Entity level method
    • Becomes a property of entities from the dataclass
      Called using an entity's reference
      The keyword this refers to the entity
