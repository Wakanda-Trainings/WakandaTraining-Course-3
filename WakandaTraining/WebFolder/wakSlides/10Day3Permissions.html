Dataclass Permissions

  • Describe - Metadata describing the dataclass is restricted to the
    assigned group; only affects the WAF (browser)
  • Read - Accessing entities is restricted to the assigned group;
    implies Describe permission, affects server and WAF.
    Error is thrown at access time if no permission.
  • Update - Saving modified entities is restricted to the assigned
    group; implies Read permission, affects server and WAF.
    Error is thrown at save time if no permission.
  • Create - Saving new entities is restricted to the assigned
    group; affects server and WAF. Error is thrown at
    save time if no permission.
  • Remove - Deleteing entities is restricted to the assigned
    group; implies Read permission, affects server and WAF.
    Error is thrown at delete time if no permission.

Attribute Permissions

  • Read - Accessing values from the attribute is restricted to the assigned
    group; only affects the WAF. Returns null as the attribute's value and no
    error is thrown if no permission
  • Update - Modifying the value of the attribute for existing entities is
    restricted to the assigned group; implies Read permission,
    only affects the WAF. Error is thrown if no permission.
  • Create - Assigning the value of the attribute for new entities is
    restricted to the assigned group; only affects the WAF.
    Error is thrown if no permission.

Method Permissions

  • Execute - Executing the method is restricted to the assigned
    group; affects server and WAF. Error is thrown if no permission
  • Promote - While executing the method, temporarily promote
    the user's permissions up to the assigned group

Other Points

  • Control points in the data model that are not assigned a group are
    left 'wide open'.
  • Only one Group can be chosen per control point. But since groups can
    contain groups, you can build a permissions hierarchy anyway
    you want, with no ambiguity in permissions.
  • You will probably want to create a group with no users to assign as
    the default for data model level permissions.
