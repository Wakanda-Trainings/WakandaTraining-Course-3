A datasource between them

One of the key points of Wakanda’s datastore is that it’s very, very fast. In our tests, we have found that an item can be queried from among a billion objects in a matter of milliseconds.

This efficiency comes from the architecture of the datastore itself: It contains all the data as well as the model, describing the datastore classes that define how the data is structured.

Custom Login


onRestrictingQuery:function()
   //create an empty collection. Return this by default.
   var result = ds.PTO_Request.createEntityCollection();
   
   //Load User entity.
   var myCurrentUser = currentUser(); 
   var sessionRef = currentSession(); 
   
   if (currentSession().belongsTo("Administrator")) {
      result = ds.PTO_Request.all();
   } else {
      result = ds.PTO_Request.query("requestor.login = :1", myCurrentUser.name);
   }
   
   return result;
}
